Debounce, 마지막 이벤트만 처리하기

Mar 2, 2023 | Mar 2, 2023
1 min read

마지막 이벤트가 발생하고 개발자가 지정한 시간 이후에 Callback을 실행하는 이벤트 처리 방식이다.
첫 이벤트가 발생할 때 타이머를 지정, 이후 이벤트가 발생할 때마다 타이머를 갱신한다.
더 이상 이벤트가 발생하지 않아 타이머를 갱신하지 않게 되면, 마지막으로 발생한 이벤트의 타이머가 끝날 때 Callback을 실행한다.

let debounceTimeoutId: number;

function handleEventDebounce(ms: number) {
  if (debounceTimeoutId !== undefined)
    clearDebounceTimeout();

  debounceTimeoutId = setTimeout(() => {
    // do something
    
    clearDebounceTimeout();
  }, ms);
}

function clearDebounceTimeout() {
  clearTimeout(debounceTimeoutId);
  debounceTimeoutId = undefined;
}

첫 이벤트가 발생할 때 타이머를 지정, 이후 이벤트가 발생할 때마다 이벤트를 무시한다.
